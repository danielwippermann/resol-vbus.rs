# resol-vbus.rs

A Rust library for processing RESOL VBus data.

[Documentation](https://docs.rs/resol-vbus/)


## Usage

First, add this to your `Cargo.toml`:

```toml
[dependencies]
resol-vbus = "0.3"
```
