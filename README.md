# resol-vbus.rs

A Rust library for processing RESOL VBus data.


## Usage

First, add this to your `Cargo.toml`:

```toml
[dependencies]
resol-vbus = "0.1"
```

Next, add this to your crate:

```rust
extern crate resol_vbus;
```
